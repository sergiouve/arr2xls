# arr2xls
Node.js CLI tool to export arrays to xlsx files.

### Installation

```
npm install arr2xls
```

### Usage
```
arr2xls -i in-file.php -o out-file.xlsx -a array_name
```

## Current support

- [x] PHP
- [ ] Javascript
- [ ] Python
- [ ] Ruby
