# xls-converter
Node.js CLI tool to export arrays to xlsx files.

## Installation

```
npm install xls-converter -g
```

## Usage
```
xls-converter -i in-file.php -o out-file.xlsx -a array_name
```

## Current support

- [x] PHP
- [ ] Javascript
- [ ] Python
- [ ] Ruby
- [ ] Java
